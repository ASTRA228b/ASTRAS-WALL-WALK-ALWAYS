## What Is It

Astras Wall Walk Always is a fun wall walk mod for gorilla tag that can be used as a walk assist

# How To use

Hold Both Grips (at the same time)

press A to set speed
press B to reset speed

it will also tell you on start

## Requirements

* BepInEx 5.x
